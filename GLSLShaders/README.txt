
GLSL Shaders
This project shows some GLSL shader examples, please refer to the .vx (vertex shader) and .fg (fragment shader) files for revelant code.


Keys:

Left-Click         : Move camera
Right-Click        : Zoom camera
F1, F2, F3.... F7  : Change shader
1, 2, 3...         : Change texture
PageUp             : Change model
Q, A               : Twist model (Shader 5)
